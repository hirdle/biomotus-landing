2. Какие данные мы собираем

В процессе использования Сервиса мы можем собирать следующие персональные данные:

  • Контактные данные: имя, номер телефона, адрес электронной почты
  • Данные для доставки: адрес доставки коляски
  • Данные аккаунта: логин, пароль (в зашифрованном виде)
  • Данные об использовании: история заказов, предпочтения по тарифам
  • Технические данные: IP-адрес, тип устройства, версия операционной системы
  • Геолокационные данные: местоположение для отображения ближайших колясок на карте (только с вашего согласия)

3. Цели обработки данных

Мы обрабатываем ваши персональные данные для следующих целей:

  • Предоставление услуг аренды электрических колясок
  • Обработка и выполнение заказов
  • Организация доставки колясок по указанному адресу
  • Связь с вами по вопросам заказов и технической поддержки
  • Улучшение качества Сервиса и пользовательского опыта
  • Отправка уведомлений о статусе заказа
  • Предотвращение мошенничества и обеспечение безопасности

4. Правовые основания обработки

Мы обрабатываем ваши персональные данные на следующих правовых основаниях:

  • Исполнение договора: обработка необходима для выполнения договора аренды
  • Согласие: вы дали согласие на обработку данных при регистрации
  • Законные интересы: обработка необходима для улучшения Сервиса
  • Юридические обязательства: обработка требуется по закону

5. Хранение и защита данных

Мы принимаем все необходимые организационные и технические меры для защиты ваших персональных данных от несанкционированного доступа, изменения, раскрытия или уничтожения:

  • Шифрование данных при передаче (SSL/TLS)
  • Безопасное хранение паролей с использованием хеширования
  • Ограничение доступа сотрудников к персональным данным
  • Регулярное обновление систем безопасности
  • Резервное копирование данных

Персональные данные хранятся в течение срока, необходимого для достижения целей обработки, но не менее срока, установленного законодательством РФ.

6. Передача данных третьим лицам

Мы можем передавать ваши персональные данные третьим лицам только в следующих случаях:

  • Курьерские службы: для организации доставки колясок
  • Платёжные системы: для обработки платежей (данные карт не хранятся на наших серверах)
  • По требованию закона: при получении официального запроса от государственных органов

Мы не продаём и не передаём ваши персональные данные третьим лицам в маркетинговых целях.

8. Файлы cookie

Наш Сервис может использовать файлы cookie для улучшения пользовательского опыта. Cookie — это небольшие текстовые файлы, которые сохраняются на вашем устройстве.

Мы используем следующие типы cookie:

  • Необходимые: для корректной работы Сервиса
  • Функциональные: для запоминания ваших предпочтений
  • Аналитические: для анализа использования Сервиса

Вы можете отключить cookie в настройках браузера, однако это может повлиять на функциональность Сервиса.
